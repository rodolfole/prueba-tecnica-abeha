## Cloning the repository

```shell
git clone https://github.com/rodolfole/prueba-tecnica-abeha.git
```

## Parte 1 (Página comparativa de autos)

Install the dependencies:

```sh
$ pnpm i
# or
$ yarn
# or
$ npm i
```

Start the app

```sh
$ pnpm dev
# or
$ yarn
# or
$ npm run dev
```

## Built With

- Reactjs
- Typescript
- Tailwind
